Important FAQ

what is the Document Object Model (DOM)?

programming interface that represents the structure of a web document in memory.

How do you select an element from the DOM?

getElementById() getElementsByClassName() querySelector()

What is event delegation in the context of the DOM, and why is it useful?

technique where you attach a single event listener to a parent element, allowing it to handle events triggered on any of its child elements, instead of adding separate listeners to each child element individually;

How do you manipulate an element's attributes and styles using the DOM?

the onfocus event handler method is used to change the element's style when the user clicks inside it
